## Project Overview

This project analyzes an HR dataset using Python to understand employee trends, performance, attrition, salary distribution, and other workforce insights. The analysis helps identify patterns that can support HR decision-making.

## Objectives
- Analyze employee data
- Find factors affecting employee attrition
- Study salary and department trends
- Visualize HR insights using charts and graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Information

The dataset contains employee-related information such as:

- Employee ID
- Age
- Department
- Salary
- Job Role
- Years at Company
- Attrition
- Performance Rating

## Analysis Performed
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Attrition Analysis
- Salary Distribution Analysis
- Department-wise Employee Analysis
- Correlation Analysis
- Data Visualization
- Visualizations

## The project includes:

- Bar Charts
- Pie Charts
- Heatmaps
- Histograms
- Count Plots
